# Netzwerk-Grundlagen

Dieses Projekt enthält meine Notizen über Computernetzwerke.

## Was ist ein Netzwerk?

Ein Netzwerk verbindet mehrere Geräte miteinander, damit sie
Informationen austauschen können.

Beispiele für Geräte in einem Netzwerk:
- Computer
- Router
- Server
- Smartphones

## LAN

LAN bedeutet "Local Area Network".

Ein LAN ist ein Netzwerk in einem kleinen Bereich,
zum Beispiel in einem Haus, einer Schule oder einer Firma.

## WAN

WAN bedeutet "Wide Area Network".

Ein WAN verbindet mehrere Netzwerke über große Entfernungen.
Das Internet ist das größte WAN.

## IP-Adresse

Jedes Gerät in einem Netzwerk hat eine IP-Adresse.

Eine IP-Adresse identifiziert ein Gerät im Netzwerk.

Beispiel:
192.168.1.1
